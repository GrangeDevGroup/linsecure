Capabilities

System Hardening

Kernel sysctl, UFW, fail2ban, AppArmor, ClamAV, rkhunter.

CORE

Paranoid Firewall

Deny-all mode. Outgoing locked to HTTPS/HTTP.

FIREWALL

Hardware Kill Switches

Blacklist camera, mic, BT, Wi-Fi at kernel level.

HARDWARE

Tor Routing

Transparent proxy + Tor Browser.

ANONYMITY

Privacy Stack

Brave, Tor, OnionShare, MAT2, DNSCrypt.

SOFTWARE

Full Restore

Revert all changes with one command.

RECOVERY
